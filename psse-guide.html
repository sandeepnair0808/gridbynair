PSSE 35 Python Integration Guide

This page shows how to integrate Siemens PSSE 35 with Python on a Windows machine. Make sure you are using the right bit version of Python (32-bit or 64-bit) depending on the PSSE version.

Python Script

import sys, os

# Inputs
psseversion = 35

# Get Python version
pythonversion = '{}.{}'.format(sys.version_info[0],sys.version_info[1]) 

# Load psspy path and environment
if psseversion == 35:
    if pythonversion in ['2.7','3.7','3.8','3.9']:
        pyver_short = int(pythonversion.replace('.', ''))
        sys.path.append(r"C:\Program Files\PTI\PSSE35\35.3\PSSPY{}".format(pyver_short))
        os.environ['PATH'] = r"C:\Program Files\PTI\PSSE35\35.3\PSSPY{};".format(pyver_short) + os.environ['PATH']
    else:
        print('Unsupported python version')
        sys.exit()
    import psse35

elif psseversion == 34:
    # Similar structure...

# Final setup
import psspy
psspy.psseinit(buses=150000)

How to Use

  1. Install compatible Python (preferably 3.7 or 3.9 for PSSE 35).
  2. Update the correct path to PSSPYxx for your Python version.
  3. Ensure `psspy.psseinit()` is used to initialize the API before loading or simulating cases.
  4. Run the script using your terminal or IDE (Admin rights may be required).
